Actividad 4

Diseña un programa en JavaScript que simule un sistema de autenticación de usuario para un sitio web. El programa debe solicitar al usuario que ingrese un nombre de usuario y una contraseña. Debe verificar las siguientes condiciones para permitir el acceso: El nombre de usuario debe ser "admin". La contraseña debe ser "secreta" o "123456". El programa debe utilizar los operadores lógicos "y" (&&) y "o" (||) para evaluar estas condiciones y permitir el acceso si el nombre de usuario y la contraseña son correctos.
