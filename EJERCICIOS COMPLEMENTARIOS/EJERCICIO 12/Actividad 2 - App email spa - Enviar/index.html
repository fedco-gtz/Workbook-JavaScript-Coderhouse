Actividad 2 - App email spa - Enviar

Renderizar una interfaz que permita realizar la simulación del envío de un correo electrónico a cierta dirección. Con la intención de manejar un respuesta del servidor, se propone realizar una petición POST con AJAX a la API JSONPlaceholder (jsonplaceholder.typicode.com/posts). Una vez confirmada la petición, instanciar un objeto Email, y asociarlo a un array de emails, efectuando una salida de confirmación de envío al usuario.
