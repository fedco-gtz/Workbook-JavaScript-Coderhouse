Actividad 2 - Provincias Argentinas

Solicitar el listado de provincias de la Argentina a la API Georef, usando una pretición AJAX (apis.datos.gob.ar/georef/api/provincias). Luego, generar un select donde los values de las etiquetas option sean el ID de cada provincia obtenida. Por último, escuchar el evento change sobre el select, asegurando que la función manejadora dispare una salida en el HTML, especificando la provincia seleccionada.
